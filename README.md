## Aplicación de Ejemplo con CamaraX
Sencilla aplicación para la implementación de la cámara mediante CamaraX.
## Librerías utilizadas
- ViewModel (Jetpack
- Navigation (Jetpack)
- CameraX (Jetpack)
## Carácterísticas
- Naveación entre fragmentos.
- Gestión de permisos.
- Captura de imágenes.
- Guardado de imágenes en caché.
